INTELLIJ IDEA PLUGIN

让注解布局回归秩序

Annotation Tidy 不试图改变你的编码习惯， 它只是把那些原本拥挤、堆叠、容易打断视线的注解前缀， 整理成更清晰、更克制，也更耐看的样子。

下载插件 查看效果

适配 IntelliJ IDEA 2025.2。为那些真正在意代码观感的人而生。

Before


@SuppressWarnings("unchecked") @Deprecated @Override
public String value() {
    return "";
}
→

After


@Override
@Deprecated
@SuppressWarnings("unchecked")
public String value() {
    return "";
}

写 Java 时，注解往往并没有错。 只是当排布失衡，整段代码也会随之显得仓促。

WHY IT MATTERS

它带来的不是复杂功能，而是一种更从容的观感

它做的事情很轻，结果却足够直接。 注解之间更有层次，声明前缀更有呼吸感， 代码也因此显得更被认真对待。

更从容 减少拥挤与堆叠，让视线停留在真正重要的地方。
更轻巧 不需要额外学习成本，一个动作就能完成整理。
更耐看 让注解块回归统一，日常阅读与维护都会更舒服。

VISUAL DIFFERENCE

变化并不喧哗，却很难忽视


class Demo {

    @Transactional @Nullable
    @SuppressWarnings("rawtypes")
    public String load() {
        return "ok";
    }

}

AFTER TIDYING

更像一段被认真维护过的代码


class Demo {

    @Nullable
    @Transactional
    @SuppressWarnings("rawtypes")
    public String load() {
        return "ok";
    }

}

对很多人来说，这并不只是格式化。 它更像是让代码重新显得体面的那一步。

INSTALL

装上它，把整理注解这件小事，交给一个更顺手的动作。

在 IntelliJ IDEA 中选择 Install Plugin from Disk...， 导入 ZIP 后即可开始使用。
